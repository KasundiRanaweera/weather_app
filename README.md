# Weather App

A simple Python weather dashboard using OpenWeatherMap and CustomTkinter.

## Run

1. Activate the virtual environment:

```powershell
c:/Users/HP/Documents/Weather_App/.venv/Scripts/Activate.ps1
```

2. Run the app:

```powershell
python main.py
```

## Project structure

- `main.py` — application entry point
- `ui/weather_app.py` — modern weather UI
- `services/weather_service.py` — OpenWeatherMap API client
- `requirements.txt` — Python dependencies
- `.gitignore` — excludes build artifacts and the virtual environment
